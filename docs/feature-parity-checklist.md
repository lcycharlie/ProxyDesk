# ProxyDesk Feature Parity Checklist

这个清单用于确保 UI 重构不会影响当前功能。

## 构建与启动

- Windows exe 可以启动并显示主界面。
- 安装包可以安装到 `Program Files`。
- 应用图标正常显示。
- 关闭窗口后进入系统托盘。
- 托盘左键可恢复窗口。
- 托盘右键可退出程序。

## 线路配置

- 可以手动输入上游代理 `host:port:user:pass`。
- 支持 HTTP 上游。
- 支持 SOCKS5 上游。
- 本地协议可选 HTTP/HTTPS。
- 本地协议可选 SOCKS5。
- 监听地址使用本机内网 IP。
- 本地端口按设置范围生成。
- 已占用端口不会重复出现。

## 转发列表

- 可以新增多条转发配置。
- 每条配置可以独立启动。
- 每条配置可以独立停止。
- 可以停止全部配置。
- 可以删除选中配置。
- 运行中的多端口可同时对外提供代理。
- 测试选中出口时只测试当前选中的那条线路。

## 供应商 API

- API 地址可配置。
- 国家参数默认 `region`。
- 城市参数默认 `st`。
- 可以选择国家/地区。
- 可以选择该国家下的城市/州。
- 选择“全部城市”时不传城市参数。
- 获取成功后自动加入转发列表。
- 获取成功后提示用户去转发列表查看。
- 本地端口池与线路配置共享。

## 国家与城市

- 国家显示为中文名称和国家简码。
- 搜索 `US`、`JP`、`NG` 等两位国家码时只返回对应国家。
- 搜索中文国家名可返回对应国家。
- 搜索英文国家名可返回对应国家。
- 国家变化后城市/州列表同步刷新。

## 系统代理

- 可以启用 Windows 系统代理。
- 可以关闭 Windows 系统代理。
- 系统代理使用当前选中的转发线路。
- 开启失败时显示错误信息。

## 出口检测

- 可以检测当前环境出口。
- 可以检测上游代理出口。
- 可以检测选中转发出口。
- 检测成功显示 IP、国家、地区、城市。
- 检测失败显示错误并写入日志。

## 日志

- 运行日志自动追加。
- 运行日志可以滚动查看历史。
- 运行日志可以清理。
- API、启动、停止、检测错误都会写日志。
